## 🛠️ How to Use This Repository

1. **Obsidian Integration**: This workspace is optimized for [Obsidian](https://obsidian.md). Double-brackets (`[[WikiLinks]]`) and relative markdown links are fully supported.
2. **Daily Routine**: Open `Pre-Market & Post-Market Trading Routine` at the start of your day, establish your `Daily Bias`, and use the `A+ Trading Setup Checklist` before entering any trade.
3. **Journal Weekly**: Review all setups, document them in your `Trading Journal`, and log weekly performance reviews in `Trading Reflection & Review Notes`.
